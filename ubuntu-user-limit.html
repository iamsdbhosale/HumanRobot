The number of users you can create on an Ubuntu (or any Linux) system is primarily limited by system constraints, not by a hardcoded number.

Each user has a UID (User ID) which is typically a 32-bit integer. The maximum is:

2^32 - 1 = 4,294,967,295

UID 0 is reserved for root, and some lower UIDs are reserved for system users.

So in theory: You can have over 4 billion users.
